# README

環境構築方法

① 　 Github 上にあるリポジトリをクローンする。※下記のコマンドをターミナルで叩く。

git clone https://github.com/shtrack/rails-docker.git

② 　クローン完了後、Docker Desktop を起動し、下記のコマンドをターミナルで叩く。

docker-compose up

③ 　 ② が無事エラーなく完了後、「http://localhost:3000」をブラウザに入力し、Rails のアプリケーションが起動していることを確認する。
問題なく起動していれば、環境構築完了である。

以上。
